# Cloud Security Engineer Technical Assessment

This repository contains my submission for the Cloud Security Engineer technical assessment.

The assessment covers:

1. Secure GCP Infrastructure with Terraform
2. Cloud Security Check
3. Automated Security Checks with Go
4. Automated Security Checks with Python
5. Third-Party Dependency Security Audit

## Design Principles

The solutions prioritize:

- Least privilege
- Secure defaults
- Infrastructure as Code
- Repeatable validation
- Automation
- Clear security reasoning
- Practical vulnerability management
